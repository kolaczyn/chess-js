# Chess

## About

Chess. Made by me.
For now you can only play against 

## Installation
```
git clone https://github.com/kolaczyn/chess-js.git
cd chess-js
npm install
npm build
```

## Technology used
I've written it in HTML, SCSS and vanilla JavaScript. I've used Webpack to compile it from CommonJS into one file.
